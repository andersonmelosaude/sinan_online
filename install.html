Instalar AutoFill (arraste para a barra de favoritos)

Arraste o botão abaixo para a sua barra de favoritos (ou clique com o botão direito e "Adicionar favorito"):

AutoFill

Instruções rápidas

  1. Abra aqui e arraste o botão "AutoFill" para a barra de favoritos.
  2. Abra o site alvo e clique no favorito.
  3. O painel aparecerá; clique "Preencher".

Importante: personalize inject.js editando o objeto FIELD_MAP no repositório.
